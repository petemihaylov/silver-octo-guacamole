## 🚀 Getting Started

### 📦 Dependencies

- next 14.1+
- node v22.0+
- npm v10.2+
- tailwind v3.3+

### 👉 Install Dependencies

```bash
npm install
```

### 👉 Development Command

```bash
npm run dev
```

### 👉 Build Command

```bash
npm run build
```

### 👉 Build and Run With Docker

```bash
docker build -t silver-octo-guacamole .
docker run -p 3000:3000 silver-octo-guacamole
```
